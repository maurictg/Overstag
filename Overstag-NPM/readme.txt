Dit is de source van Materialize CSS aangepast voor overstag. 
Het bevat ook een nodejs server met de volgende functionaliteiten

1. Het compilen van de sass naar een css bestand
2. Het compilen van het less bestand in de Overstag/wwwroot/less map naar overstag.css
3. Het compilen van hQuery in mapje public
4. Het minimizen van alle bestanden

Om de server te gebruiken, voer de eerste keer "npm i" uit

Om de sass te compilen: npm run sass
Om de less te compilen: npm run less
HQuery compilen en alles minifyen: npm run compile
